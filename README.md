# Django address book

the task of this project is to connect to the Flexibee demo JSON file, which contains a list of companies and their detailed information. This information must be stored in a database and then sent via API to the frontend, which will be built on Vite + React.

- [Django-rest-framework](https://www.django-rest-framework.org/)
